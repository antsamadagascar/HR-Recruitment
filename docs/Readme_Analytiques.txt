- application comptabilite analytique sur le RIZ 

Pourquoi ? 
- Obtenir un seuil de rentabilite 

- Comment ? 
  En calculant:
  - cout par centre : 
    - ADM/DIST 
    - USINE 
    - PLANTATION 
    =>    - cout achat  = 
                Prix d'achat unitaire * qte achetee + frais annexe (transport,stockage , ...)

          - cout de production  = 
                cout des matieres premieres + cout de main d'oeuvre 

          - cout de vente  ou cout de distribution =
                frais de marketing + frais de distribution + communication sur vente

  - cout de revient ( global | moyenne sur chaque produit ) : 
    - cout d'achat + cout de production + cout distribution 

  - cout total ( periodique ) integralite des depenses d'une entreprise sur une periode :
    - cout fixe + cout variable 

  - Seuil de rentabilite si et seulement si : 
    - marge sur cout de vente == cout fixe 
    or marge sur cout de vente = Chiffre d'affaire - cout variable ( C.A - C.V )

    formule de depart : 
      - marge global = C.A - (cout Variable + cout fixe)                // C.A : Chiffre d'Affaire
      avec C.A = ensemble vola ananan'le olona @ hafa sy ananany eny aminy 
            
            C.A  = prix de vente * Qte vendue 
  
  - Centre structurel : 
      - EXEMPLE : departement RH , comptabilite 
      Support ou gestion de l'entreprise 
        - ne participe pas directement a la production 
        - ROLE : soutenir les activites operationelles en garantissant une infrastructure et des processus efficaces 
        (AKA CERVEAU)
            
  
  - Centre operationnel : 
      - EXEMPLE : Atelier de production  , Equipe de vente , entrepot 
      Responsable des operations de base de l'entreprise telles que la fabrication , la production ou la distribution 
        - implication direct dans la production 
        - ROLE : genere des revenus pour l'entreprise 
        (AKA CORPS)
